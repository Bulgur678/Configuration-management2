# Configuration-management2

Вариант 13. Мирошкин Дмитрий ИКБО-70-24 
Разработан инструмент анализа и визуализации графа зависимостей пакетов Alpine Linux. Реализация выполнена в виде 
CLI-приложения на Python без использования готовых менеджеров пакетов или сторонних библиотек для получения информации 
о зависимостях.

Инструмент поддерживает два режима работы: с удаленным репозиторием и с тестовым файлом. В режиме удаленного репозитория 
осуществляется загрузка и анализ APKINDEX.tar.gz для извлечения информации о зависимостях. В тестовом режиме 
используется файловое представление графа зависимостей с пакетами, обозначенными заглавными латинскими буквами.

Основной функционал включает построение графа зависимостей с учетом транзитивности с применением алгоритма BFS с 
рекурсией, обработку циклических зависимостей, поиск обратных зависимостей

Инструмент принимает параметры через командную строку, включая имя анализируемого пакета, URL-адрес репозитория или путь
к тестовому файлу, режим работы. 
Реализована полная обработка ошибок для всех входных параметров.

Результаты каждого этапа разработки сохранены в системе контроля версий в соответствии с принятыми стандартами 
оформления коммитов. Инструмент обеспечивает полный цикл анализа зависимостей - от получения данных до их визуального 
представления.
